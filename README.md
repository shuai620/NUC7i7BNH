+ OS: macOS Catalina 10.15.7 (Build 19H2) x86_64 / iMac16.2

+ CPU: Intel® Core™ i7-7567U Processor (4M Cache, up to 3.90 GHz)

+ RAM: 8GB x 2 DDR4 2133 MHz
